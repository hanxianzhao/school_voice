# 小小班 · 污水处理厂互动流程图（前端单页）

一个纯前端静态页面，面向小朋友讲解污水处理的全过程：从“产生 → 管道 → 厂内处理 → 清水排放”，支持点击节点查看解释，带动态“水流”与“污泥”动画，移动端适配。

## 使用方式

1. 直接双击打开 `index.html`（或在本目录启动一个本地服务器）。
2. 左侧控制区：
   - “开始流水 / 暂停流水”：控制动画播放/暂停
   - “重播”：重置动画进度
3. 点击图里的任意方框节点，可在左侧看到对应解释。

## 文件结构

```
index.html   # 页面结构与SVG流程图
styles.css   # 页面与SVG样式，含水流/污泥动画
app.js       # 交互脚本（播放控制、点击节点、信息面板）
README.md    # 使用说明
```

## 自定义

- 可在 `index.html` 中调整各节点矩形坐标和连接路径（`path`）。
- 可在 `styles.css` 修改动画速度（`@keyframes flow/sludge` 或 stroke-dasharray/dashoffset）。
- 可在 `index.html` 的每个节点 `<g class="node step" ...>` 上通过 `data-title` 与 `data-text` 修改讲解文案。

## 适配

- 支持移动端：小屏幕下布局自动变为上下排布。
- 无任何构建依赖，适合课堂现场/家长演示。
